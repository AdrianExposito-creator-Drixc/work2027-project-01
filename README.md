# Work2027 – Project 01 (Análisis de Ventas)

## 📌 Descripción
Proyecto básico de análisis de datos que:
- Lee un archivo CSV de ventas
- Calcula métricas (total ventas, items)
- Agrupa ventas por producto
- Exporta resultados a archivo
- Incluye versión profesional con pandas
- Genera visualización (gráfico)

---

## ⚙️ Tecnologías
- Python 3
- pandas
- matplotlib
